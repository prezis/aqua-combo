# Contributing

1. Fork the repo
2. Make your change
3. Run tests: `python3 -m pytest tests/ -v`
4. Submit a PR

### What to contribute
- New GPU tier configs in `examples/`
- Improvements to `scripts/gpu_detect.py` (new GPU vendors, better detection)
- SKILL.md phase improvements (better prompts, new patterns)
- New agent personas in `agents/`
- Bug fixes

### Rules
- No hardcoded paths, model names, or API keys in SKILL.md
- Tests must pass
- Keep SKILL.md under 400 lines
